# Agent Spec

## Overview
A RAG (Retrieval-Augmented Generation) application built using ADK 2.0's Graph-based Workflow API. The application answers medical questions based exclusively on the MedQuAD dataset stored in a pre-built Chroma database (`./medquad_chroma`).

## Assumptions
- The database in `./medquad_chroma` is already populated and can be queried using `Chroma` and `OpenAIEmbeddings` with `text-embedding-3-small`.
- `OPENAI_API_KEY` and `GEMINI_API_KEY` are loaded from `.env`.
- Gemini 1.5 Flash (via Gemini API) is used for the generation node.
- The workflow runs locally in prototype mode.

## Example Use Cases
- **Valid query**: User asks "What is pyridoxine-dependent epilepsy?". The agent queries Chroma, retrieves the document, and replies with the definition citing the source (e.g. `3_GHR_QA`).
- **Unanswerable query**: User asks "What is the capital of France?". If the question is not related to medical space, do not query the database and answer "I am anwering only medical related questions". Otherwise, the agent queries Chroma, finds no relevant context, and replies with "there is no source."


## Tools / Nodes Required
- **Chroma Query / Retrieval Node**: Queries the local Chroma SQLite database using `OpenAIEmbeddings(model="text-embedding-3-small")`.
- **Generation Node**: Formulates a response using Gemini 1.5 Flash.

## Constraints & Safety Rules
- **Strict Source Limitation**: The model MUST base its answer ONLY on the retrieved documents from the MedQuAD database.
- **Fall-back Behavior**: If the retrieved documents do not contain the answer, or if similarity scores/relevance is below threshold, or if no relevant context is retrieved, the model must reply with exactly "there is no source.".
- **Source Citation**: Every answered query must include a citation of the source folder (the `source` metadata field in the Chroma documents).

## Success Criteria
- Retrieve relevant context for MedQuAD-related questions.
- Respond with "there is no source." for non-medical or out-of-dataset questions.
- Include the exact source metadata name (e.g., `3_GHR_QA`) in the response.
